1. Deployment Process
   
It was quite easy to use. I didn't have any problems when deploying and it took care of prettyy much everything. I had to make a couple small adjustments to dockerfile and prisma.config.ts but I'm not sure if thats because I wasnt set up properly or if it was requirements with fly.io. Regardless, it was very easy and I didnt have any questions about it.

2. Pros and Cons

It runs full docker containers so anything can be deployed which is nice. You have more control over the infrastructure. It also runs PostGres. Overall it was very easy to deploy.

Its a smaller ecosystem compared to Vercel and pricing isnt great.

3. Challenges and Surprises

It was very easy to deploy which was a surprise but the issue I did have took me a minute to figure out. Like I said in the Deployment process, I had to make some adjustments to DockerFile and the Prisma config file but other than those small changes, it was quite easy.
