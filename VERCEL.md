## 1. Deployment Process Overview

Deploying to Vercel was super easy! All I had to do was connect my GitHub to Vercel and then select the repository I wanted to connect/deploy. From there I just had to input the environment variables into the Vercel project and I was good to deploy. I then had to ensure that I allowed the Google OAuth to accept the Vercel link into the app. Then we could make changes to the repo and it would automatically re-deploy all of our changes.
